# Sistema de Verificação de Pares e Ímpares

## 📖 Descrição

Este projeto foi desenvolvido em Java como exercício de prática da linguagem. O programa solicita ao usuário que informe cinco números inteiros, armazena os valores em um vetor (`array`) e realiza as seguintes operações:

- Conta a quantidade de números pares.
- Conta a quantidade de números ímpares.
- Calcula a soma de todos os números informados.
- Exibe os números digitados utilizando `Arrays.toString()`.

## 🚀 Conceitos praticados

- Variáveis
- Entrada de dados com `Scanner`
- Arrays (`int[]`)
- Estruturas de repetição (`for`)
- Estruturas condicionais (`if/else`)
- Operador módulo (`%`)
- Contadores
- Acumulador
- Classe `Arrays`


## 🎯 Objetivo

Praticar os conceitos fundamentais da linguagem Java, como entrada de dados, manipulação de arrays, estruturas de repetição, condicionais e operações matemáticas.
